A) Description - top

Onova is a creative, modern and clean Figma design which is designed for agencies, freelancers, photographers, videographers and production studio. It contains modern featured areas which can fit any type of portfolio agencies. The template designed using bootstrap 5 and 1320px container, thats why its fully responsive on any device and you can convert it easily to Html, WordPress or any CMS which you want. The Figma file is perfectly organized, so You could easily customize everything. Feel free to get support as we love to help our clients. If you like my Figma templates, please rate it 5 stars.
